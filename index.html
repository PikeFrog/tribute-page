Frog

For this project I began with creating the mesh of a frog using Autodesk Mudbox. Following this I was able to bring this mesh over into Maya and prep the mesh for a rig. Upon finishing the rig of the frog I animated a simple jump. This video contains the actual model that a user would want as well as the control curves and the rig itself.

Sneak-Walk Cycle

This project utilized the base humanoid mesh from Auodesk Mudbox. This mesh was brought into Maya and was rigged. From here the rig was animated with in Maya as well and brought over to Adobe AfterEffects to capture both the mesh and the rig.
